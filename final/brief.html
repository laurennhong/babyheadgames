☆ final project brief ☆

Summary

For my final project I will be revising Studio 3: Game On! to function more smoothly and to be visually more enticing.

Project Goals

I would like to revise three aspects of the design. Firstly, I would like to change and add movement to my animations. For this, I will change my still illustrations into gifs. This will require me to also change the javascript to switch between the .gif version of my illustrations to the .jpg versions. I also want to make the transitions between mouse interactions smoother. Fixing the sudden layout shifts during game time. Lastly, I want to edit the javascript so that there are more audio options. Currently, you can only turn on the audio in my game, however, I want to make it so you can turn it off as well.

Design Strategies

Visually, I want to stay on par with my creepy horror theme. I will keep the typeface and color palette the same as the original design, however, I want to make alterations to the animations. As soon as you open the site, the two creepy twins are displayed on opposite sides of the screen. For my new design, I want to make the twins move by creating a gif. The movement will be simple but creepy to stay with the horror theme. Once you begin the game I will switch to the jpg version of the illustration. I also plan on making slight adjustments to font sizing and layout for better readability and to add stars to the background for a night time aesthetic.

Interaction Strategies

For interactions, I plan to make the transition at the start of the game smoother. Once you press ‘start game’ the screen will ease into the the game. There is also a glitch that occurs when users roll the dice in the game. This glitch interrupts the user experience, therefore, I want to edit the CSS to get rid of the glitch and make for a more professional game. I also would like to have an animation to reward the winner of the game. I believe this will make the game more exciting for users and better indicate when the game is over. I will acheive this using CSS animations.

Experience Goals

I hope to emulate a niche horror theme that is aesthetically pleasing and representative of my illustration style. I want to have fun with the design and further my Adobe Illustrator skills. I also want to challenge myself to better my understanding of CSS animations and transitions and javascript as a whole. Using online resources has helped me throughout this process, but I want to understand every function I am coding. I believe a stronger understanding of CSS animations and transitions will make my future coding projects more professional. I want to pay attention to the details in my game’s layout to optimize its readability making the user experience exciting and easy.
